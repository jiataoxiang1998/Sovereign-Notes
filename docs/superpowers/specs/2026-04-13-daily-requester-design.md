# 每日工作总结助手 - 设计规范

## 1. 项目概述

### 项目名称
每日工作总结助手 (Daily Requester)

### 项目类型
Electron + Vue 3 + Vite 桌面应用

### 核心功能
- 记录每日待办事项、已完成事项、发现问题、无法解决的问题
- 一键生成每日工作总结并复制到剪贴板
- 查看历史记录

### 目标用户
需要每日记录工作内容和生成工作总结的职场人士

## 3. 功能规范

### 3.1 核心功能

#### 待办事项
- 添加新待办：标题、优先级（高/中/低）
- 标记完成：点击复选框移动到已完成
- 删除：勾选后批量删除

#### 已完成事项
- 显示：标题、完成时间、原始优先级
- 删除：勾选后批量删除

#### 发现的问题
- 添加：标题、描述、严重程度（严重/一般/轻微）
- 删除：勾选后批量删除

#### 无法解决的问题
- 添加：标题、描述、严重程度
- 删除：勾选后批量删除

#### 生成每日总结
- 点击按钮生成并复制到剪贴板
- 格式：
```
2026-04-13 每日工作总结
========================

【待办事项】
- [高] 完成项目规划
- [中] 代码审查

【已完成事项】
- [完成 14:30] 需求分析

【发现问题】
- [严重] 数据库连接超时

【无法解决的问题】
- [严重] API文档缺失
```

### 3.2 数据存储

- 存储位置：%APPDATA%/daily-requester/data/
- 格式：JSON文件
- 文件命名：YYYY-MM-DD.json
- 结构：
```json
{
  "date": "2026-04-13",
  "todos": [
    {
      "id": "uuid",
      "title": "内容",
      "priority": "high",
      "createdAt": "2026-04-13T09:00:00Z"
    }
  ],
  "completed": [],
  "issues": [],
  "blockers": []
}
```

### 3.3 历史记录

- 侧边栏显示历史日期列表
- 点击日期加载对应数据
- 支持查看任意历史日期

### 3.4 系统集成

- 开机自启动
- 窗口最小化到托盘
- 自定义标题栏（拖动、关闭、最大化）
